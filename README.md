# Chastushki Research Project

Этот каталог содержит исследование и продуктовые артефакты по идее сервиса генерации русских частушек.

## Текущая структура

### Основные документы

1. `01_market_scan.md`
   - обзор аналогов, конкурентов, смежных продуктов и исследовательских референсов
   - отвечает на вопрос: что уже есть на рынке и насколько занята ниша

2. `02_technical_plan.md`
   - техническое проектирование сервиса без реализации
   - описывает возможный pipeline, подход к генерации, аудио-слой, риски по качеству, правам и safety

3. `03_summary.md`
   - короткая сводка по всей идее
   - объясняет, почему идея выглядит жизнеспособной и где у неё лучший продуктовый угол

4. `04_design_doc.md`
   - основной продуктовый design-doc
   - содержит позиционирование, аудитории, JTBD, MVP scope, user flows, IA, UX-принципы, монетизацию, GTM-гипотезы и риски

5. `05_prd_mvp.md`
   - главный PRD-документ для MVP
   - содержит цель MVP, scope, user stories, требования, acceptance criteria, метрики, release criteria, rollout assumptions и открытые вопросы

6. `06_founder_memo.md`
   - краткий стратегический документ на 1-2 страницы
   - полезен для обсуждения идеи как стартап-гипотезы: opportunity, wedge, why now, risks, moat, что валидировать первым

### Дополнительные рабочие документы

- `product_strategy.md`
  - стратегическая часть продуктового проектирования
  - фокус на positioning, ICP, JTBD, wedge и differentiation

- `product_experience.md`
  - UX-ориентированная часть продуктового проектирования
  - фокус на user flows, экранной структуре, форматах выдачи, loop'ах улучшения, монетизации и GTM

### Архив промежуточных материалов
